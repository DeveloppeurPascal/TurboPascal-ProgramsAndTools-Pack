## Installation des codes sources

Si vous téléchargez un fichier ZIP depuis GitHub il sera vide. Vous devez utiliser la commande "git" ou une interface utilisateur dédiée pour cloner ou dupliquer ce dépôt et ses sous-modules.

Après avoir cloné le dépôt n'oubliez pas de mettre à jour les projets liés avec cette commande :

```bash
git submodule update --remote
```

Si vous n'avez pas cloné le projet avec ses sous-modules vous devrez saisir les commandes suivantes :

```bash
git submodule init
git submodule update --remote
```

[Ce dépôt de code](https://github.com/DeveloppeurPascal/TurboPascal-ProgramsAndTools-Pack) et tous ses sous-projets sont disponibles sur [mon compte GitHub](https://github.com/DeveloppeurPascal).

## Comment demander une nouvelle fonctionnalité, signaler un bogue ou une faille de sécurité ?

Si vous voulez une réponse du propriétaire de ce dépôt la meilleure façon de procéder pour demander une nouvelle fonctionnalité ou signaler une anomalie est d'aller sur [le dépôt de code sur GitHub](https://github.com/DeveloppeurPascal/TurboPascal-ProgramsAndTools-Pack) et [d'ouvrir un ticket](https://github.com/DeveloppeurPascal/TurboPascal-ProgramsAndTools-Pack/issues).
